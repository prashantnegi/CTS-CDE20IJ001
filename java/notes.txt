Rules for naming identifiers

1. No keywords
2. No special chars other than _ $
3. Cannot start with number 
4. Case senitive

Conventions

1. Class name: Upper camelcase

    TraineeDetails
    TraineePerformanceData


2. Variables/functions: Lower camel case

    createTrainee()

    assignNewTrainer()

    traineeName

    traineeEmail

3. Constants: Fully upper case

    MAX_XP
